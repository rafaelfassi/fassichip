The FassiChip is a free and open project. However, its development, maintenance, and constant updates involve costs and many hours of hard work. If the FassiChip is useful for you, help keep this project alive by making a donation.


You can also help the project in the following ways:
  • Creating new drivers;
  • Creating and improving your software documentation;
  • Translating the software and/or documentation into other languages;
  • Reporting bugs and making suggestions for improvement of the software;
  • Disseminating the software on websites, forums, blogs, download sites, etc...;
  • Creating tutorials and/or video-tutorials to encourage use by new users;
  • Creating graphic art, such as: icons, logo, etc...;
  • If you have advanced knowledge of C++ programming, work will not miss!
