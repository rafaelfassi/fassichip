O FassiChip é um projeto livre, aberto e gratuito. Porém, seu desenvolvimento, manutenção e constantes atualizações envolvem custos e muitas horas de trabalho duro. Se o FassiChip é útil para você, ajude a manter este projeto vivo fazendo uma doação.


Você também pode ajudar o projeto das seguintes formas:
  • Criando novos drivers;
  • Criando e aprimorando a documentação do software;
  • Traduzindo o software e/ou documentação para outros idiomas;
  • Reportando bugs e fazendo sugestões para melhoria do software;
  • Divulgando o software em sites, fóruns, blogs, sites de downloads, etc...;
  • Criando tutoriais e/ou vídeo-tutoriais para incentivar o uso por novos usuários;
  • Criação de arte gráfica, como: ícones, logotipo, etc...;
  • Se você possuir conhecimentos avançados de programação C++, trabalho não vai faltar!
